# firmware

## Which firmware can be used

- Marlin
- Klipper
- RRF

## What does the board need that it can be used?

- 32bit
- 6 Stepper Motors or 5 Stepper + 1 Toolboard
- Neopixel Support
- 3 Thermistor Inputs (Hotend, Heated Bed, Enclosure)
- min 3 fan outputs
